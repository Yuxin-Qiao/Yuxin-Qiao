## Open source

I contribute to [steipete/CodexBar](https://github.com/steipete/CodexBar), a macOS menu-bar app for tracking AI coding-provider usage.

My contributions focus on provider usage parsing, quota visibility, billing semantics, diagnostics, privacy-aware redaction, menu-bar reliability, and regression testing.

Selected work:

* **Provider integration**: MiniMax quota visibility, token-plan parsing, and usage-card regression tests.
* **Billing correctness**: failed-record filtering, reset-window separation, and spend normalization.
* **Diagnostics**: redacted export flows for safer issue reporting and debugging.
* **Reliability**: snapshot preservation and edge-case tests for network and menu-bar failures.
